## Getting Started

\*Install dependencies:

```bash
npm install
# or
yarn
```

\*Run development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tests:

After load development server, in the other terminal run:

```bash
yarn cypress open
```
